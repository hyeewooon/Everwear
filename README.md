# EVERWEAR

## Project Stack

- React
- Redux
- Node.js
- Express JS
- MongoDB

## Start Development

### Install Dependancy Library

```
npm install or npm i (inside the root directory)
```

### Start Development Server

```
npm start or npm run start (inside the client directory)
```
